# NIH ExPORTER Scorecard Schemas

**Last Updated:** 2025-11-25  
**Version:** 1.0

## Overview

Four comprehensive scorecards provide multi-dimensional views:

| Scorecard | Rows | Granularity | Purpose |
|-----------|------|-------------|---------|
| grant_scorecard | ~513K | CORE_PROJECT_NUM | Grant details with linked outputs |
| pi_scorecard | Variable | PI name | Career metrics and productivity |
| institution_scorecard | Variable | Institution | Organizational portfolios |
| ic_scorecard_enhanced | ~120 | IC code | Institute/Center comparisons |

See full documentation in repository for detailed schemas and examples.
